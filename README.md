# 🖋️ Dear Data Estúdio: Proposta Visual & Infografia Narrativa de Consistência Mental

> **Um projeto interativo de jornalismo de dados humanizados, inspirado no método artístico de Giorgia Lupi e Stefanie Posavec, desenvolvido especialmente para uma estudante de jornalismo em Manaus, AM.**

Este projeto é uma ferramenta interativa e de alta fidelidade visual para documentar, analisar e expressar graficamente o paradoxo entre a **rigidez da consistência química** (tratamento continuado de saúde mental) e a **fluidez das variações emocionais diárias**.

O infográfico foi desenhado sob a premissa de que *"tomo remédios todos os dias, mas cada dia é um universo único e diferente por dentro"*.

---

## 🎨 A Metáfora Visual de Dados (Como ler o Infográfico)

Cada elemento da folha de desenho (Canvas) representa uma variável de dados traduzida em geometria artesanal:

*   **O Grid de Fundo (1180 pontos)**: Representa os **1.180 dias de tratamento continuado ininterrupto** (Pregabalina + Antidepressivo) acumulados desde janeiro de 2022. Os pontos cinzas são o histórico silencioso; os pontos coloridos e pulsantes no final representam a semana atual.
*   **Diâmetro do Círculo Central**: Mapeia o nível de **Energia & Carga Mental** do dia (Escala de 1 a 5).
*   **Semicírculo Esquerdo**: Representa a cor e a emoção predominante da **Manhã**.
*   **Semicírculo Direito**: Representa a cor e a emoção predominante da **Noite**.
*   **Anel Externo Tracejado**: Indicador químico. Aparece apenas quando a **medicação foi tomada corretamente (Tratamento OK)**.
*   **Círculo Interno Verde Escuro**: Indica a atividade de rotina (**Dia de Trabalho** ativo em Manaus).
*   **Linha Diagonal com Âncora**: Representa um **Evento Extra** marcante no dia. O comprimento da linha é proporcional ao impacto emocional do evento (Leve, Médio ou Forte).

## 💻 Funcionalidades do Estúdio Interativo

O estúdio conta com uma interface escura ultra-premium inspirada em editores profissionais de design (como Figma e Canva Studio):

1.  **Aba `Diário` (Configurador da Semana)**:
    *   **Adição Dinâmica**: Adicione novos dias com um clique. A ferramenta calcula cronologicamente a próxima data (ex: `24/05`, `25/05`) e gera os dados correspondentes.
    *   **Controles Deslizantes e Accordion**: Altere sentimentos, cores, energia e eventos de cada dia de forma intuitiva com efeitos visuais de sanfona.
    *   **Presets Rápidos de Paleta**: Aplique instantaneamente esquemas de cores profissionais desenvolvidos por designers: **Pastel Suave**, **HSL Vibrante** ou **Monocromático**.
    *   **Exclusão Prática**: Exclua registros com um clique (com confirmação segura).
2.  **Aba `Conceito`**:
    *   Contextualização editorial, científica e artística do projeto para apresentação acadêmica ou exposições.
3.  **Aba `Canva` (Régua de Posicionamento Proporcional)**:
    *   Gera um manual descritivo com as coordenadas exatas convertidas para a proporção padrão de stories/cartazes (**1080x1920px**) do Canva. Ideal para que ela possa recriar a arte física no Canva com precisão milimétrica. Possui botão "Copiar manual" com um clique.
4.  **Aba `Editorial` (Redação Acadêmica Automatizada)**:
    *   Redige um ensaio crítico-jornalístico formal analisando os dados inseridos na semana atual em tempo real. Ideal para utilizar como descrição conceitual da infografia:
5.  **Folha de Desenho 3D (Canvas Preview)**:
    *   Incline e movimente a folha tridimensionalmente ao passar o mouse sobre ela.
    *   Passe o cursor sobre os círculos do infográfico para ler os sentimentos e dados exatos através de **tooltips interativos**.
    *   Botão para alternar entre o estilo **Dear Data (Artesanal riscado à mão)** e **Modern Tech (Vetor limpo moderno)**.
    *   Botão **"Régua Canva"** para sobrepor linhas guias físicas de coordenadas e eixos X/Y diretamente no papel.
6.  **Exportação em Vetor**:
    *   Botão para baixar a arte diretamente como arquivo **SVG de alta qualidade**, com background creme sólido e filtros de textura física preservados para impressão de cartaz ou publicação.

## 🛠️ Como Executar e Usar

A aplicação foi estruturada usando apenas tecnologias nativas da web (HTML5, CSS3 avançado e JavaScript puro), o que significa que **não requer nenhuma instalação de dependências ou servidores backend**.

1.  Baixe a pasta com os arquivos do projeto.
2.  Dê um duplo clique no arquivo **`esboço_final_infografia.html`** para abri-lo em qualquer navegador de internet moderno (Google Chrome, Microsoft Edge, Safari, Firefox, Opera, etc.).
3.  **Persistência Automática**: Todas as edições de dias, cores e textos que você fizer são guardadas automaticamente no banco de dados local do seu próprio navegador (`localStorage`). Você pode fechar o navegador e voltar no dia seguinte que seu rascunho continuará intacto!
4.  Para limpar tudo e voltar ao modelo original de exemplo de sua amiga, basta clicar no botão **"Resetar Rascunho"** no topo da tela.
